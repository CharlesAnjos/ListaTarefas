# ListaTarefas (Projeto Final PDMI)
Atividade de lista de tarefas da disciplina de PDMI do Prof. Fagno.
Este projeto é o projeto final da disciplina.\
A proposta implementar funcionalidades extras ao trabalho de Listas de Tarefas. As funcionalidades extras são:
- armazenar e listar afazeres por categorias
- implementar lista de favoritos independente de categorias
- implementar declaração de afazer como completo e listagem de afazeres completos independente de categoria
